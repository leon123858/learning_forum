###### learning_forum
# 成大社科院論壇系統(SSF)結案報告
## 功能概觀
- 使用者
  - 帳號相關
    - 登入註冊
    - 驗證碼登入
  - 電子郵箱
    - 帳號認證
    - 密碼救援
  - 個人資料相關
    - 個資修改
    - 我的最愛
    - 我的通知
  - 論壇形式
    - 公有論壇
    - 私有論壇
  - 論壇功能
    - 發文與修改發文
    - 留言閱讀與留言
    - 貼文閱讀與篩選
- 管理者
  - 一般管理者
    - 看版貼文屏蔽
    - 看版資訊管理
    - 看板資料下載
  - 核心管理者
    - 創建新看板
    - 所有看版貼文屏蔽
    - 所有看版資訊管理
    - 所有看板資料下載
